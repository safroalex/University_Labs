A Mathematical Software Library


P. A. Jacobs, N. J. Lott & J. Day
peterj@mech.uq.edu.au

This collection is an amalgamation of the CMATH C Software Library (written by Peter and Nigel) and the DMATHPAK FORTRAN library written by John Day together with some more recently written routines that we have found useful.

  1. Introduction
  2. Content by Subject Area
  3. Content by Function Name
  4. CMATH (C) Routines
  5. Recent Routines in C
  6. DMATHPAK (FORTRAN) Routines
  7. Recent Routines in FORTRAN

This collection is made available as a hypertext document primarily to save me having to continually copy and loan disks. The documentation is adapted from the Chiwriter files for CMATH and is presently in a state of disarray. It will be updated occasionally.
